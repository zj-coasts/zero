 ## 目录结构
 ```
 |- node_modules: 依赖包
 |- public: 公共资源文件（不需要被js编译的文件）
 |- src: 项目主目录
    |- api: 接口文件
    |- assets: 资源文件
        |- iconfont: 字体图标
    |- components: 公共组件
    |- config: 项目配置项
        |- http: http 配置文件
    |- router: 路由
    |- store: 全局状态管理
    |- styles: 样式文件
    |- views: 视图

 ```
